# Markdown

## Markdown 是什么

> Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们「使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档 」—— [维基百科](https://zh.wikipedia.org/zh-hans/Markdown)
